# 기사 분류 기준 (Classification Rules)

> 최종 수정: 2026-03-22
> 대상 파일: `backend/crawler.py`, `backend/classifier.py`

---

## 전체 흐름

```
네이버 뉴스 API
      ↓
  [크롤러] 키워드로 수집 + 24h→7일 이내 기사만
      ↓
  [분류기] 제목·본문 키워드 매칭으로 섹션 확정
      ↓
  [pipeline] 기타 제외 + 중복 제외 후 DB 저장
```

> ⚠️ **핵심 주의사항**: 크롤러가 수집하지 않은 기사는 분류기가 존재조차 모릅니다.
> 분류가 안 된다면 분류기 문제보다 **크롤러 키워드 누락**일 가능성이 높습니다.

---

## 분류 우선순위 (순서가 곧 규칙)

분류기는 아래 순서대로 검사하고, **처음 매칭된 규칙에서 즉시 반환**합니다.

```
1순위  제목에 무관 패턴 → 기타 (즉시 제외)
2순위  제목에 당사 계열사명 → 섹션3_당사관련
3순위  제목에 경쟁사명 + 위기키워드 → 섹션2_반면교사
4순위  제목에 경쟁사명 (위기 없음) → 섹션4_경쟁사관련
5순위  제목+본문에 동일 업계 2개사 이상 → 섹션1_유통트렌드
6순위  제목+본문에 트렌드/정책 키워드 → 섹션1_유통트렌드
7순위  그 외 → 기타 (DB 저장 안 함)
```

---

## 섹션별 상세 기준

### 🔴 섹션3\_당사관련

**조건**: 기사 **제목**에 당사 계열사명이 포함된 경우

**당사 계열사 키워드** (제목에 하나라도 있으면 매칭):
| 키워드 | 비고 |
|--------|------|
| 현대지에프홀딩스, 현대백화점그룹 | 그룹 |
| 현대백화점, 현대백, 현대百 | 백화점 |
| 현대홈쇼핑 | 홈쇼핑 |
| 현대아울렛, 현대프리미엄아울렛, 현대면세점 | 유통 |
| 현대리바트, 리바트, 지누스, 현대L&C | 리빙 |
| 한섬 | 패션 |
| 현대그린푸드, 현대바이오랜드 | 식품 |
| 현대퓨처넷, 현대에버다임, 현대이지웰, 현대벤디스 | 기타 계열사 |

**점수 배정**:
- 위기키워드 동시 등장 → `relevance_score: 8.0`, 중요도: **상**
- 일반 언급 → `relevance_score: 6.0`, 중요도: **중**

### 🟠 섹션2\_반면교사

**조건**: 기사 **제목**에 경쟁사명 AND (제목+본문)에 위기키워드 1개 이상

**경쟁사 키워드**:
| 그룹 | 키워드 |
|------|--------|
| 롯데 | 롯데백화점, 롯데百, 롯데마트, 롯데쇼핑, 롯데홈쇼핑, 롯데온 |
| 신세계 | 신세계백화점, 신세계百, **신세계**, 이마트, SSG닷컴 |
| 갤러리아/AK | 갤러리아백화점, 갤러리아百, AK플라자, AK百 |
| GS | GS리테일, GS홈쇼핑, GS25, GS샵 |
| CJ | CJ온스타일, CJ올리브영, 올리브영 |
| 기타 | 홈플러스, 쿠팡, 무신사, 위메프, 11번가, G마켓, 옥션, 티몬 |

> ⚠️ "신세계"는 단독 키워드라 **"신세계마켓"도 매칭**됩니다 → 위기키워드 없으면 섹션4로 분류됩니다.

**위기 키워드** (제목+본문 기준):
| 카테고리 | 키워드 |
|----------|--------|
| 윤리/갑질 | 갑질, 불공정거래, 허위광고, 과대광고, 담합, 과징금, 횡령, 배임, 탈세, 리베이트, 납품단가, 공정위 제재, 공정위 조사, 소비자 기만 |
| 안전/위생 | 식중독, 이물질, 위생불량, 제품결함, 유해물질, 리콜, 불량제품, 안전사고, 부상자 발생 |
| 개인정보 | 개인정보 유출, 해킹 피해, 정보유출, 고객정보 유출 |
| 사회논란 | 논란, 불매, 사과문 발표, 직장내괴롭힘, 갑을 논란 |
| 소비자신뢰 | 가품, 위조품, 짝퉁, 가품 판정, 정품 위조, 품질 논란, 소비자 피해, 환불 거부, 배송 지연 논란, 구매 피해 |

**세부 카테고리** (위기키워드 종류로 자동 결정):
- `반면교사_개인정보` — 개인정보 유출, 해킹
- `반면교사_안전위생` — 식중독, 이물질, 리콜
- `반면교사_소비자신뢰` — 가품, 위조품, 환불 거부
- `반면교사_사회논란` — 논란, 불매, 사과문
- `반면교사_갑질윤리` — 위 어디에도 안 걸릴 때 기본값

**점수 배정**: `min(7 + 위기키워드 매칭수, 10)` → 8점 이상이면 중요도 **상**

### 🟣 섹션4\_경쟁사관련

**조건**: 기사 **제목**에 경쟁사명이 있지만 **위기키워드는 없는** 경우
(보도자료, 신규 서비스, 실적 발표, 전략 기사 등)

- `relevance_score: 5.0`, 중요도: **하**
- `is_competitor: False` (위기 사례가 아니므로)

**예시**:
- ✅ "신세계마켓 1주년, 강남점 핵심 콘텐츠로 자리" → 섹션4
- ✅ "쿠팡, 로켓배송 새벽 서비스 확대" → 섹션4
- ❌ "쿠팡은 정품, 무신사는 가품 판정" → **가품** 키워드로 섹션2

## 알려진 한계 및 주의사항

| 문제 | 원인 | 현황 |
|------|------|------|
| 경쟁사 보도자료가 섹션4 대신 기타로 빠질 수 있음 | 크롤러 SECTION4_KEYWORDS에 없는 기업/키워드 | 키워드 추가 필요 |
| 중요도 "상" 기사가 적음 | 섹션3은 위기키워드 필요, 섹션2는 8점 이상 | 구조적 한계 |
| "신세계" 단독 키워드가 신세계마켓·신세계건설 등 모두 매칭 | 부분문자열 매칭 방식 | 허용 범위로 판단 |
| 동일 기사가 섹션1과 섹션4 기준 모두 해당 시 | 2순위인 섹션3→섹션2→섹션4 먼저 처리됨 | 우선순위 정책 유지 |
